python3-模块学习-3-urllib和requests

urllib和requests练习：

用requests库来索取html内容：
requests.get(url) 是response格式（requests.models.Response）
requests.get(url).content 是bytes格式；
requests.get(url).text 是str格式。

对urllib库而言，
request.urlopen(url) 是response格式（http.client.HTTPResponse）；
request.urlopen(url).read() 是bytes格式
request.urlopen(url).read().decode(‘utf-8’) 是str格式
