python爬虫-博士学位分析-1

前面发的是喜闻乐见的抓美女图片的代码，原因很简单：

  • 网站结构好分析，甚至没有用到bs4、scrapy之类的爬虫常用工具;
  • 图片抓下来就可以是最终成果，给自己学习编程一个阶段性的收获;

  • 也可以用PIL等工具进行后续分析处理.

    通过抓两个图片网站熟悉了urllib、os、re和multiprocessing几个工具的基本用法和python程序的基本结构之后，找一个有意义的内容进行抓取分析，让爬虫变的有意义。

    实际选择的是某大学的博士生情况。看看如今博士延期的具体比例。原理很简单，通过抓取所有博士生在线答辩公告，对比学号和答辩日期。同时，实际上也可以得到很多其他有意思的信息。由于

    学校公告不会一直保留，所以第一步是把答辩公告全部抓到本地，并不断增量更新。有前面的爬虫练手，这个抓取是很好写的。
